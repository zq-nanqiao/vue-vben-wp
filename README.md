## 简介

Vue Vben wp 是基于Vben5的一个学习项目。

## 特性

- **最新技术栈**：使用 Vue3/vite 等前端前沿技术开发
- **TypeScript**: 应用程序级 JavaScript 的语言
- **主题**：提供多套主题色彩，可配置自定义主题
- **国际化**：内置完善的国际化方案
- **权限** 内置完善的动态路由权限生成方案

## 安装使用

- 获取项目代码

```bash
git clone https://github.com/vbenjs/vue-vben-admin.git
```

- 安装依赖

```bash
cd vue-vben-admin

corepack enable

pnpm install
```

- 运行

```bash
pnpm dev
```

- 打包

```bash
pnpm build
```
